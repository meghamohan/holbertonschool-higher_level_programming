The `2-matrixDiv` module
=============================

First step is to import ``matrixDiv`` from the 
``2-matrixDiv`` module:

>>> matrixDiv = __import__('2-matrix_divided').matrix_divided

Now you can start using it

>>> matrixDiv([[1, 2, 3], [4, 5, 6]], 3)
[[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]

>>> matrixDiv([[-100]], -50)
[[2.0]]
